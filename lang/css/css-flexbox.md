# CSS Flexbox

## Center contents
```css
parent {
  display: flex;
  justify-content: center;  /* horizontal center */
  align-items: center;      /* vertical center */
}
```

## Vertical line
```css
parent {
  display: flex;
  flex-direction: column;
}
```

Reference: https://css-tricks.com/snippets/css/a-guide-to-flexbox/
